# Changelog

## 1.0.0

Initial release of the Antigravity plugin for Claude Code.

- `/antigravity:review` — run a standard code review via `agy`
- `/antigravity:adversarial-review` — run an adversarial review via `agy`
- `/antigravity:rescue` — delegate tasks to `agy`
- `/antigravity:status` — check background job status
- `/antigravity:result` — retrieve a completed job result
- `/antigravity:cancel` — cancel an active background job
- `/antigravity:setup` — check `agy` availability and authentication
